Badge Creator
==============
 the create_badge is a Python script that allows you to create circular badges from images. 
 the verify_badge is a python script  that verifies whether an input image meets the criteria to be used as a circular badge. 

Features
=========
Verify that the input image is of size 512x512 pixels.
Ensure that the non-transparent pixels are within a circular boundary.
Check if the colors in the badge give a "happy" feeling (configurable).
Optionally, check if the colors in the badge give a "happy" feeling.

Prerequisites
=============
Pillow library (a Python Imaging Library fork)
